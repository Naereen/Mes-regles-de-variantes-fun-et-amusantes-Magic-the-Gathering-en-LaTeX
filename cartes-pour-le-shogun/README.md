## Qu'est-ce que le mode de jeu alternatif en EDH - Shogun ?

C'est un mode de jeu alternatif à Magic: the Gathering, à rôles cachés, pour changer et varier un peu lors de parties de Magic en EDH Commander multi-joueurs.

Le Shogun se veut une simplification mode « Treachery » qui est présenté et disponible sur [le site officiel MTGtreachery.net](https://mtgtreachery.net/fr/#howto).
